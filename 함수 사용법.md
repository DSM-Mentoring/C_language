함수의 기능 : 특정한 기능을 하는 코드들을 묶어 하나의 명령어처럼 사용 가능 하게 해줍니다

2. 함수 이름 : 함수를 호출하기 위한 이름을 명시

​    

3. 매개변수 목록 : 함수 호출 시에 전달되는 인수의 값을 저장할 변수들을 명시

​    

4. 함수 몸체 : 함수의 고유 기능을 수행하는 명령문의 집합
